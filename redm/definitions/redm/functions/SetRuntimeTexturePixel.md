Sets a pixel in the specified runtime texture. This will have to be committed using `COMMIT_RUNTIME_TEXTURE` to have any effect.

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `tex` | `number` | A handle to the runtime texture. |
| `x` | `number` | The X position of the pixel to change. |
| `y` | `number` | The Y position of the pixel to change. |
| `r` | `number` | The new R value (0-255). |
| `g` | `number` | The new G value (0-255). |
| `b` | `number` | The new B value (0-255). |
| `a` | `number` | The new A value (0-255). |

### Returns
