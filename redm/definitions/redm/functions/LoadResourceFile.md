# Function: LoadResourceFile()

## Call Signature

```ts
function LoadResourceFile(resourceName, fileName): string;
```

Defined in: [src/common-game/definitions/redm.d.ts:6267](https://github.com/nativewrappers/nativewrappers/blob/696dfbbc9d1d7665dd81166c0f1b4bee71d2b3f4/src/common-game/definitions/redm.d.ts#L6267)

Reads the contents of a text file in a specified resource.
If executed on the client, this file has to be included in `files` in the resource manifest.
Example: `local data = LoadResourceFile("devtools", "data.json")`

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `resourceName` | `string` | The resource name. |
| `fileName` | `string` | The file in the resource. |
