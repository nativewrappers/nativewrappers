# Function: CreateRuntimeTextureFromImage()

## Call Signature

```ts
function CreateRuntimeTextureFromImage(
   txd, 
   txn, 
   fileName): number;
```

Defined in: [src/common-game/definitions/redm.d.ts:1397](https://github.com/nativewrappers/nativewrappers/blob/686d3e4293666c68edfc1e42776f4b1c55b0605b/src/common-game/definitions/redm.d.ts#L1397)

Creates a runtime texture from the specified file in the current resource.

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `txd` | `number` | A handle to the runtime TXD to create the runtime texture in. |
| `txn` | `string` | The name for the texture in the runtime texture dictionary. |
| `fileName` | `string` | The file name of an image to load. This should preferably be a PNG, and has to be specified as a `file` in the resource manifest. |

### Returns

`number`

A runtime texture handle.
