# Function: SetHandlingField()

## Call Signature

```ts
function SetHandlingField(
   vehicle, 
   class_, 
   fieldName, 
   value): void;
```

Defined in: [src/common-game/definitions/redm.d.ts:22205](https://github.com/nativewrappers/nativewrappers/blob/8fd2e00d7e048b9268f6c990b83eef252e4d6150/src/common-game/definitions/redm.d.ts#L22205)

Sets a global handling override for a specific vehicle class. The name is supposed to match the `handlingName` field from handling.meta.
Example: `SetHandlingField('AIRTUG', 'CHandlingData', 'fSteeringLock', 360.0)`

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `vehicle` | `string` | The vehicle class to set data for. |
| `class_` | `string` | The handling class to set. Only "CHandlingData" is supported at this time. |
| `fieldName` | `string` | The field name to set. These match the keys in `handling.meta`. |
| `value` | `number` | The value to set. |

### Returns

`void`
