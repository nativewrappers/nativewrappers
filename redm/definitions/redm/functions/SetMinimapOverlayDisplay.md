Defined in: [src/common-game/definitions/redm.d.ts:22567](https://github.com/nativewrappers/nativewrappers/blob/8fd2e00d7e048b9268f6c990b83eef252e4d6150/src/common-game/definitions/redm.d.ts#L22567)

Sets the display info for a minimap overlay.

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `miniMap` | `number` | The minimap overlay ID. |
| `x` | `number` | The X position for the overlay. This is equivalent to a game coordinate X. |
| `y` | `number` | The Y position for the overlay. This is equivalent to a game coordinate Y, except that it's inverted (gfxY = -gameY). |
| `xScale` | `number` | The X scale for the overlay. This is equivalent to the Flash \_xscale property, therefore 100 = 100%. |
| `yScale` | `number` | The Y scale for the overlay. This is equivalent to the Flash \_yscale property. |
| `alpha` | `number` | The alpha value for the overlay. This is equivalent to the Flash \_alpha property, therefore 100 = 100%. |

### Returns

`void`
