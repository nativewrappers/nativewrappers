# Function: SetHandlingFloat()

## Call Signature

```ts
function SetHandlingFloat(
   vehicle, 
   class_, 
   fieldName, 
   value): void;
```

Defined in: [src/common-game/definitions/redm.d.ts:22235](https://github.com/nativewrappers/nativewrappers/blob/5084b61ea79d43bdf84faea5c688d0b87ec5e48b/src/common-game/definitions/redm.d.ts#L22235)

Sets a global handling override for a specific vehicle class. The name is supposed to match the `handlingName` field from handling.meta.
Example: `SetHandlingFloat('AIRTUG', 'CHandlingData', 'fSteeringLock', 360.0)`

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `vehicle` | `string` | The vehicle class to set data for. |
| `class_` | `string` | The handling class to set. Only "CHandlingData" is supported at this time. |
| `fieldName` | `string` | The field name to set. These match the keys in `handling.meta`. |
| `value` | `number` | The floating-point value to set. |

### Returns

`void`
