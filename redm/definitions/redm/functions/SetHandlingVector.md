# Function: SetHandlingVector()

## Call Signature

```ts
function SetHandlingVector(
   vehicle, 
   class_, 
   fieldName): void;
```

Defined in: [src/common-game/definitions/redm.d.ts:22293](https://github.com/nativewrappers/nativewrappers/blob/8fd2e00d7e048b9268f6c990b83eef252e4d6150/src/common-game/definitions/redm.d.ts#L22293)

Sets a global handling override for a specific vehicle class. The name is supposed to match the `handlingName` field from handling.meta.
Example: `SetHandlingVector('AIRTUG', 'CHandlingData', 'vecCentreOfMassOffset', vector3(0.0, 0.0, -5.0))`

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `vehicle` | `string` | The vehicle class to set data for. |
| `class_` | `string` | The handling class to set. Only "CHandlingData" is supported at this time. |
| `fieldName` | `string` | The field name to set. These match the keys in `handling.meta`. |

### Returns

`void`
