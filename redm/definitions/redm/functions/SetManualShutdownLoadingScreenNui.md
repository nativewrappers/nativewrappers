# Function: SetManualShutdownLoadingScreenNui()

## Call Signature

```ts
function SetManualShutdownLoadingScreenNui(manualShutdown): void;
```

Defined in: [src/common-game/definitions/redm.d.ts:22483](https://github.com/nativewrappers/nativewrappers/blob/8fd2e00d7e048b9268f6c990b83eef252e4d6150/src/common-game/definitions/redm.d.ts#L22483)

Sets whether or not `SHUTDOWN_LOADING_SCREEN` automatically shuts down the NUI frame for the loading screen. If this is enabled,
you will have to manually invoke `SHUTDOWN_LOADING_SCREEN_NUI` whenever you want to hide the NUI loading screen.

### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `manualShutdown` | `boolean` | TRUE to manually shut down the loading screen NUI. |

### Returns

`void`
